First post

This is my first blog post (template shamelessly stolen from Niklas). Starting is always the step that takes the longest for me, and since I'm a huge noob at html/css I wanted to use a template to get started faster. I figure if I'm consistent, over time this blog will become more original.

What will the website do?

Well for starters, this website is NOT to showcase my front-end dev skills. For now, I want to start some sort of a daily blog to get me in the habit of consistently pushing to github and coming back to my personal projects. When the website is "done" I see it having some basic features:

  • A blog for me to write whatever I want
  • An about section, for me to describe who I am
  • A "projects" section, where I can share fun little projects I'm working on, similar to the blog but with more code.

That's all for now folks!
